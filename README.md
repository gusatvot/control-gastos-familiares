# 💰 Control de Gastos Familiares

Aplicación web para gestionar gastos e ingresos familiares.

## 🚀 Características

- ✅ Autenticación de usuarios
- ✅ Gestión de gastos e ingresos  
- ✅ Análisis por categorías
- ✅ Reportes detallados
- ✅ Multi-usuario por familia

## 🛠️ Tecnologías

- React 18
- Tailwind CSS
- Supabase (PostgreSQL + Auth)
- Lucide Icons

## 📦 Instalación

1. Clonar repositorio
2. Abrir `index.html` con Live Server
3. Configurar Supabase

## 🔧 Configuración

Configurar variables de Supabase en `index.html`
